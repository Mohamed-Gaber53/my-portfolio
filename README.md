Portfolio Website
A modern and responsive portfolio website built with React and Vite.
🚀 Features

Responsive design
Contact form with EmailJS integration
Smooth animations
Modern UI/UX

🛠️ Technologies

React
Vite
EmailJS
SweetAlert2

🔗 **[Live Demo](https://mga-portfolio.netlify.app/)**



📦 Installation
bash# Clone the repository
git clone <your-repo-url>

# Navigate to project directory
cd portfolio

# Install dependencies
npm install
⚙️ Configuration
Create a .env file in the root directory:
envVITE_API_KEY=your_emailjs_public_key
🚀 Running the Project
bash# Development mode
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
📧 Contact Form Setup

Create an account on EmailJS
Create an email service
Create an email template
Copy your Public Key to .env file
Update service ID and template ID in Contact.jsx
